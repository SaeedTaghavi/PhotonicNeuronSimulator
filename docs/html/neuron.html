Neuron¶

class neuron.Neuron(params={})[source]¶

This is a Neuron object Initialize it as myneuron = Neuron() # default, or myneuron = Neuron(params) where params is a dict with parameter values, for example params = {‘model’: “FitzHughNagumo”, ‘y0’: [0.0, 0.0]} params = {‘model’: “Yamada0”, ‘mpar’: {‘P’: 0.8, ‘gamma’: 1.e-2, ‘kappa’: 1, ‘beta’: 1e-3} }

steady_state(yguess=None)[source]¶

solve for the no-input steady state of the neuron. Choose yguess “wisely” to avoid unphysical roots. Recommended to test steady state by ruling solve(np.zeros(N)) and seeing if state stays stationary.
